06_AS_常用快捷键

Sep 8, 2017

Android studio 常用快捷键

  • Ctrl + O 重写方法 @override
  • Alt + Insert 重构方法
  • Ctrl + Shift + L 格式化代码
  • Alt+Enter：导入包，快速修复。
  • Alt+1 打开project目录
  • Alt+方向键 切换工程文件上标签页
  • Alt+shift+方向键 切换工程文件下标签页

  • Ctrl+/：当行注释，反注释再按一次即可。

  • Ctrl+Shift+/：模块注释，反注释再按一次即可。

  • Alt+Insert：可以生成构造器/Getter/Setter等。

  • Ctrl+Alt+V：生成方法返回值。
