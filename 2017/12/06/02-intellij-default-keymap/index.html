02_intellij_default_keymap

Dec 6, 2017

常用intellij快捷键

  • Tool Window
  • Editor Tab
  • Run
  • code
  • Navigate
  • Edit
  • View
  • File

Tool Window

  • shift+ESC 隐藏
  • F12 显示(最近激活窗口)

Editor Tab

  • Alt+左/右 切换
  • Ctrl+F4 关闭

Run

  • shift+F10 执行run

  • Ctrl+ F5 重新rerun

  • Ctrl+ F2 停止stop

  • shift+F9 调试debug

  • Ctrl+F8 断点breakpoint

  • F7 一步step_into

Code

  • Ctrl+ O Override

  • Ctrl+ I Implements

  • Alt+ insert Generate

  • Ctrl+Alt+ T Surround with

  • Ctrl+ J Insert Template

  • Ctrl+ / Comments

  • Ctrl+shift+ / Comments block

  • Ctrl+Alt+ L Format

  • Ctrl+ space 自动

  • Ctrl+Alt+ space

  • Ctrl+shift NUMPAD- 全文折叠

  • Ctrl+shfit NUMPAS+ 全文展开

  • Ctrl+ NUMPAD- 段落折叠

  • Ctrl+ NUMPAD+ 段落展开

  • Ctrl+Alt+ NUMPAD- 递归折叠

  • Ctrl+Alt+ NUMPAD+ 递归展开

  • Ctrl+shift+ 上/下 控制语句statement

  • Alt+shift+ 上/下 控制一行line

Navigate

  • Ctrl+ G 到某一行
  • Alt+ 上/下 到下个方法
  • Ctrl+ B 到声明
  • Ctrl+shift+ B 到类型声明
  • Ctrl+Alt+ B 到实现
  • Ctrl+ U 到super
  • F4 到source

View

  • Alt+1 project窗口
  • Alt+4 run窗口
  • Alt+5 debug窗口

Edit

  • Ctrl+shift+ Enter 补全代码
  • Ctrl+shift+ U 转换大小写
  • Ctrl+ D 重复选择项
  • Ctrl+ W 扩大选择项
  • Ctrl+shift+ W 缩小选择项

File

  • Ctrl+Alt+ S 设置settings
