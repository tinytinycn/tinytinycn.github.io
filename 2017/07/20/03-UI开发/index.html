5.滚动控件RecyclerView的用法

  • 布局文件加入，指定id，由于不是内置系统sdk中，需要完整的包路径。
  • 创建一个适配器XXXAdapter继承RecycleView.Adapter<XXXAdaptet.ViewHolder>
  • 定义一个内部类ViewHolder，继承RecycleView.ViewHolder。通过ViewHolder的构造函数传入一个View参数（RecycleView子项最外层的布局），从而利用View获取布局中的控件。
  • 通过XXXAdapter的构造函数，接收传入的数据。
  • 重写XXXAdapter的onCreateViewHolder()，将子布局加载进来，创建ViewHolder实例，将加载出来的布局传入构造函数中。
  • 重写onBindViewHolder()，将子项数据进行赋值，子项被滚动到屏幕中时，执行。
  • 重写getItemCount()方法，返回数据源长度。
  • 初始化数据//RecycleView实例对象//setLayoutManager()方法传入一个LayoutMannger//XXXAdapter实例对象//setAdapter()方法设置一个适配器
