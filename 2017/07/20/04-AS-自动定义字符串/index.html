Android studio 自动定义字符串

利用Android studio 的功能，自动定义字符串，帮我们把字符串定义到 strings.xml中。

  1. 首先，将光标移入字符串里面，接着，按下ALT + Enter.
  2. 选择Extract string resource.
  3. 键入名称 即可。
