02_crazyJava_数据类型

Jun 9, 2014

第二篇 Java 数据类型

1.注释

  • 单行注释“//”放在内容之前
  • 多行注释“/”开始，“/”结束
  • 文档注释“/*”开始，“/”结束，中间部分全都是文档注释，会被提取到API文档中（javadoc 命令的使用）

2.分隔符、标识符、关键字

  • ； { } [ ] ( ) . 空格 都具有特殊分割作用，通称为分隔符
  • 标识符用于给程序中变量、类、方法命名的符号。必须字母、下划线、美元符开头，后面跟任意数目字母、数字、下划线、美元符。 （Java语言区分大小写 ）
  • Java语言中具有特殊用途的单词称为关键字keyword（定义标识符时，不能与关键字相同，否则编译出错）

3.数据类型

  • Java语言是强类型语言，每个变量和表达式都有一个在编译时就确定的类型，所以所有变量必须显式声明类型
  • 所有变量必须先声明，后使用
  • 声明变量，只要指定变量的类型和变量名即可（type varName;）
  • Java语言支持的类型：基本类型 和 引用类型
  • 基本类型：数值类型 和 布尔Boolean类型，数值类型有 整数类型和浮点类型，整数类型有 byte/short/int/long/char，浮点类型有 float/double
  • 引用类型：类、接口、数组类型、null类型（null类型，没有名称，不可能声明一个null类型的变量。空引用null是null类型变量唯一的值）。引用类型就是对一个对象的引用（引用类型变量实际上就是一个指针）
  • 字符串不是基本数据类型，字符串类型是一个类，是一个引用数据类型
  • 整型 byte（1字节）/short（2字节）/int（4字节，默认）/long（8字节）
  • 整数常量 表示形式：十进制、二进制0b/0B开头、八进制0开头、十六进制0x/0X
  • 补码计算规则：正数的补码和原码相同，负数的补码是其反码＋1，反码是对原码按位取反，最高位符号位不变
  • 字符型 char（2字节）
  • 字符常量 表示形式：单个字符 ‘A’、特殊字符 ‘\n’、Unicode值 ‘\uXXXX’
  • 浮点型 float（4字节）/double（8字节，默认）
  • 浮点数 表示形式：十进制 3.14、科学计数法 3.14e2/3.14E2
  • 只有浮点类型的数值才可以使用科学计数法形式表示
  • 布尔型 boolean（1字节）
  • 布尔数值只能是 true false
  • Java 7种数值类型之间可以相互转换，类型转换方式：自动类型转换、强制类型转换
  • 当把一个表数范围小的数值或变量直接赋给另一个表数范围大的变量时，系统可以自动类型转换
  • 强制类型转换，(targetType)value
  • 表达式类型的自动提升：当一个算术表达式种包含多个数据类型时，表达式的数据类型将发生自动提升

4.运算符

  • 算数运算符 +-*/%
  • 赋值运算符 =
  • 位运算符
  • 比较运算符 ><=!
  • 逻辑运算符 && || ！
  • 三目运算符 (expression)？if-true-statement：if-false-statement
