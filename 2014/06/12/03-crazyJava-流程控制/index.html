第三篇 Java 流程控制

1.顺序结构

  • 程序总是从上向下一次执行

2.分支结构

  • if 语句

  • switch 语句

  • 控制表达式expression的数据类型只能是 byte/short/int/char四个整数类型和 枚举类型。（Java 7 允许控制表达式为String类型）

  • case 后代码块，break省略会引入一个陷阱

3.循环结构

  • 循环语句包含四个部分：初始化语句、循环条件、循环体、迭代语句

  • while 循环语句（先判断，后执行）

  • do while 循环语句（先执行，后判断）

  • for 循环语句

  • for循环条件总比循环体多执行一次

4.控制循环结构

  • Java提供continue break 来控制循环结构
  • break 用于完全结束一个循环，跳出循环体
  • break 紧跟一个标签，这个标签用于外层循环，则可以跳出外层循环
  • continue 用于结束本次循环，接着开始下一次循环
  • return 直接结束一个方法
