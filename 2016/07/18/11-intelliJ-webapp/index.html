intellJ-idea 普通Java工程创建webapp步骤

  1. create new project
  2. Java(不需要勾选template) >next >finsh
  3. Open Module Setting(F4) >Project Settings
  4. Project(默认)
  5. Modules > “+” > add Web > apply
  6. Facets > Create Artifacts > apply > ok
  7. Modules > 选中工程 > Dependencies > “+” > libaray > tomcat7.0 >apply
  8. src > new > package
  9. src > new > Servlet > 保持前缀Servlet命名(可选,勾选生成注解)
  10. web > new > jsp > index.jsp
  11. Run > Edit Configuration …
  12. “+” > Tomcat Server > local
  13. Tomcat Server > Server(默认)
  14. Tomcat Server > Deployment > “+” > Artifact > Application context: /YOURWEBAPP >apply
  15. Tomcat Server > Server > On update action: Redeploy//On Frame deactivation: Update classes and resource > apply
  16. run..(shift+F10)

注意事项

  • 第9步, 勾选生成注解annotation, web.xml不会生成相关配置信息(此时,推荐使用注解进行配置: 添加注解urlPatterns=”/ServletXXX”即可);不勾选默认web.xml生成配置相关信息.

intellJ-idea 16 创建webapp步骤(推荐)

  1. create new project
  2. JavaEE > Web Application
  3. Open Modules Settings > Modules > 选中工程 > Dependencies > “+” > libaray > tomcat7.0 > apply
  4. src > new > Servlet > 默认已勾选注解方式创建类 > 配置注解同上
  5. Run > Edit Configuration …> Tomcat Server > 配置同上
  6. run..(shift+F10)
