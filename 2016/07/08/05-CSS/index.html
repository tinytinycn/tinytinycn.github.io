元素类型

  • 块级元素
    默认,独占一行/ 可设置宽高/ 不设置宽高, 宽填满父元素, 高取决内容元素/ 可设置内外边距/
  • 行内元素
    默认,多个行内元素同处一行/ 不能设置宽高/ 可设置左右内外边距, 上下外边距设置无效/
  • 行内块元素
    既具备行内元素特征,又具备块级元素特征
