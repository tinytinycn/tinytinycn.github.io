abstract 抽象

  • 抽象的.
  • abstract 修饰方法,类

抽象方法

声明方法,但不实现方法,叫做抽象方法.

抽象类

  • 如果一个类中包含抽象方法,那么该类一定是抽象类.
  • 如果一个类是一个抽象类,那么该类 不一定 有抽象方法.
  • 将子类公用有的一些行为提取出来声明.供给子类继承实现.
  • 抽象类是一个公共的模板.

注意事项

  1. 抽象类可以包含普通方法
  2. 抽象类 不一定 有抽象方法(不推荐),此时子类继承抽象父类无意义.
  3. 抽象类不可以被实例化.
  4. 抽象类仍然包含构造方法.

abstract 和 final

不能同时修饰一个类.

abstract修饰类,抽象类总是被用于继承.final修饰类,不可变类不能被继承.二则冲突,同时修饰无意义.
