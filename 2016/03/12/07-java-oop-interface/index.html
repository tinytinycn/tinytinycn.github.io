接口

  • 一个公共的模板,一种规范.用来此之上进行扩展.
  • 接口是一种特殊的抽象类.
  • 接口只能包含常量和抽象方法.

注意事项

  1. 接口中,默认会将其中的变量添加修饰符: public staitc final
  2. 接口中,默认会将其中的方法添加修饰符: public abstract
  3. 接口中,没有构造方法(jdk1.8增加了构造方法和静态方法的支持)
  4. 一个类可以多实现多个接口.如 A implements B,C,D { … }
  5. 接口可以多继承多个接口.如 A extends B,C,D { … }
  6. 接口不可以实例化

内部类

  • 类的成员: 成员变量,方法,构造器,内部类
  • 定义在外部类中的类.

注意:

  1. 内部类和外部类方法,成员变量属于同一级别
  2. 内部类可以调用外部类中的所有成员.

匿名内部类

一个方法的参数列表位置是一个接口或抽象类类型,则会用到匿名内部类.
