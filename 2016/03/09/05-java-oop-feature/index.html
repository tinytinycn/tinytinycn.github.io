继承

  • 提高代码复用.
  • 只能有一个直接父类,可以有多个子类.
  • extends

继承的特性

  • 单继承性
  • 继承的传递性
  • 不能继承的成员: private私有成员和构造方法
  • static静态成员, 静态方法可以被继承和隐藏, 而静态方法不能够被重写, 因此不能实现多态

重写 override

  • 只能重写父类的方法.
  • 具有继承关系的两个类,子类对父类中的某个方法进行方法体的重新实现.
  • 规则: (同名同参两小一大)方法名/参数列表同,返回值类型/抛出异常 子类<=父类,访问修饰符 子类>=父类.

注意事项

向上转型,父类引用子类对象(Furit f = new Apple();)时:

  • 引用变量f 不能访问到子类特有的成员
  • 引用变量f 调用父子类共有的方法,实际上调用的 子类中的方法
  • 引用变量f 访问成员变量,能访问到的成员变量取决变量的类型.

封装

对数据和操作进行包装,同时将内部的实现细节隐藏.

static 关键字

  • 静态的.

  • static 修饰的成员是属于类的.非static修饰的成员是属于对象的.

  • static 可修饰变量,方法,代码块.

  • 非static方法, 实例方法.非staitc变量,实例变量.

  • 静态成员变量
    可以通过类名调用.静态成员变量的默认初始化随着类的加载而进行,且只执行一次.

  • 静态方法
    直接通过类名调用.(静态方法的主调是一个类,而非静态方法或成员方法的主调是一个对象)

  • 静态代码块
    该代码块随着类的加载而执行,且只执行一次.一般将公共资源放入静态代码块中.

final 关键字

  • 不可变的.

  • final 可修饰变量,方法,类.

  • 修饰变量,变量的值不可再改变.

  • 修饰方法,方法不可被重写.

  • 修饰类,类不可被继承.(String,Math类)

static final

  • 修饰变量,表示常量.

多态

  • 类型相同的父类引用,指向不同的实例对象.
  • 意义在于,扩展.
