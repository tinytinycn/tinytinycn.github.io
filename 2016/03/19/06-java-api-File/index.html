File

java.io.File 用于表示文件或目录

创建File对象

  • File(String pathName)
  • File(File parent, String child)
  • File(String parent, String child)
  • 使用File.separator常量代替分隔符
  • 创建File对象，并未真正创建文件本身

操作文件

  • createNewFile();//创建文件
  • isFile();
  • isDiretory();
  • length();
  • exists();//是否存在文件或目录
  • delete();

操作目录

  • mkdir();//创建目录

  • mkdirs();//创建多级目录

  • ListFiles();//返回指定目录下所有文件或目录对象

  • list();//返回指定目录下所有文件或目录名

  • getName();

  • getAbsolutePath();

  • FileFilter();//路径过滤器

RandomAccessFile文件操作

Java提供文件随机访问操作，该类读写操作基于指针操作。

  • RandomAccessFile(File file, String mode); //mode:”r”/“rw”
  • RandomAccessFile(String fileName, String mode);
  • write(int b);//向文件中写数据，文件不存在，则会自动创建文件
  • write(byte[] bytes);
  • write(byte[] bytes, int offset, int len);
  • read() -int
  • read(byte[] bytes) -int
  • close(); //关闭资源
  • getFilePointer(); -long //获取当前文件的偏移量，读取一个字节数据后，偏移量后移一位
  • seek(long pos); //移动到指定位置

临时文件

File.createTempFile(); -File //创建临时文件，使用完直接删除delete
