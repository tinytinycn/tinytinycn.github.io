基本数据类型的包装类wrapper

包装类是不可变类,构造出包装类对象后,不允许更改其中的值.

  • int Integer
  • long Long
  • double Double
  • short Short
  • float Float
  • byte Byte
  • char Char
  • boolean Boolean
    其中, Number是Byte/Float/Double/Short/Long/Integer 的父类

常用方法:

  • parseXXX(String s); String>>XXX 得到字符串中的某个值
  • toString(); String
  • valueOf(xxx i); XXX 得到一个值的包装类
  • xxxValue(); xxx 得到包装类中的值

自动装箱和拆箱

Java 5 加入autoboxing 功能, 依靠编译器的预处理工作.
