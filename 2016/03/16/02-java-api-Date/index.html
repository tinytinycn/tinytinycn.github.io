Mar 16, 2016

目录

  • Date
  • Calendar抽象类
  • SimpleDateFormat日期操作类

Date

构造方法:

  • Date() 构造方法创建一个Date对象，表示当前时间
  • Date(long ms)构造方法创建一个ms指向的Date对象
    方法:
  • getTime() 得到当前时间到历元差值毫秒数(long)
  • setTime(long ms)将date对象指向的ms表示的时间
  • System.currentTimeMills() 当前时间到历元差值毫秒数(long)

Calendar抽象类

获取Calender对象:

  • 创建GregoianCalendar子类对象
  • 调用Calendar.getInstance()静态方法
    常用日历字段:
  • Calender.YEAR
  • YEAR/MONTH(0-11)/DATE/DAY_OF_WEEK(周日开始1-7)/…
    方法:
  • get(字段);
  • getTime(); -Date
  • getTimeInMillis(); -long
  • set();
  • setTime(Date date);
  • setTimeInMillis(long ms);

SimpleDateFormat 日期操作类

格式转换: 指定格式的字符串的日期和Date对象之间进行转换
获取SimpleDateFormat对象:

  • new SimpleDateFormat(“格式”);
    格式:
  • yyyy-MM-dd HH:mm:ss
    2017-11-29 09:30:59
    方法:
  • parse(String s_date); 将字符串解析成日期类型
  • format(Date date); 格式化日期类型成字符串
