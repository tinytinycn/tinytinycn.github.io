emacs-evil-vim-keymap

Mar 6, 2018

以下type 键位, 均位于normal state.

1 光标移动

  1. 光标移动 hjkl
  2. 移动到buffer开头 gg
  3. 移动到buffer结尾 G
  4. 移动到下n个word [number] w
  5. 移动到下n个word的结尾 [number] e
  6. 移动到下n个word的开头 [number] b

2 插入文本 (增)

  1. 光标前插入 i
  2. 光标后插入 a
  3. 光标上一行插入 O
  4. 光标下一行插入 o
  5. 光标行首插入 I
  6. 光标行尾插入 A

3 删除文本 (删)

  1. 光标下删除一个字符 x
  2. 光标往后删除一个单词 dw
  3. 光标往后删除一行 d$ 或 D
  4. 光标下一整行 dd
  5. delete command格式:
    [number] command object 或 command [number] object
    2dw 或 d2w
    2dd 或 d2d

4 修改文本 (删改)

注意的是: 修改时, normal state >> insert state

  1. 光标下修改一个字符 r (自动esc退出insert state)
  2. 光标下连续修改 R (直到esc退出)
  3. 光标往后修改一个单词 cw (直到esc退出)
  4. 光标往后修改一个行 c$ (直到esc退出)
  5. change command格式:
    [number] command object 或 command [number] object

5 替换文本 (换改)

  1. 当前行替换一次 :s/old/new
  2. 当前行替换n次 :s/old/new/g
  3. 某几行替换一次 :s#,#/old/new/g
  4. 全部替换 :%s/old/new/g
  5. 全部替换(每次改变有提示) :%s/old/new/gc

6 搜索文本 (查)

  1. 正向搜索 / (n 下一个匹配项 N 上一个匹配项)
  2. 反向搜索 ? (n 下一个匹配项 N 上一个匹配项)
  3. 定位配对符号 % (定位(),[],{})

7 命令行

  1. 外部命令行格式 :!command
    :!ls or :!dir 列出目录
    :!rm or :!del FILENAME 删除文件
    :w FILENAME 当前buffer全部内容写出到文件FILENAME
    :#,#w FILENAME 当前buffer某行内容写出到文件FILENAME
    :r FILENAME 从文件FILENAME恢复内容到当前buffer

8 复制 粘贴

normal模式下:

  1. yw
  2. yy
  3. y$
  4. y^
  5. :reg 粘贴板
  6. “ayw 复制到a粘贴板栈中, “ap 粘贴从a粘贴板栈中.

按字符选择模式:

  1. v (visual state)
  2. hjkl
  3. y (yank)
  4. hjkl
  5. p (put)

按行选择模式:

  1. V (visual state)
  2. hjkl
  3. y
  4. hjkl
  5. p

按块选择模式:

  1. CTRL-v (visual state)
  2. hjkl
  3. y
  4. hjkl
  5. p

9 剪切 粘贴

同上 复制粘贴

  1. v (visual state)
  2. hjkl
  3. d (delete)
  4. hjkl
  5. p (put)

10 退出 保存

  1. 退出 :q
  2. 退出不保存 :q!
  3. 保存 :w
  4. 保存 另存为 :w filename.md
  5. 退出保存 :wq 或 :x 或 ZZ

11 切换模式

  1. 退回正常模式 ESC 或 CTRL-[
