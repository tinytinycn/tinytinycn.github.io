nginx-https代理设置

Oct 29, 2018

nginx https代理

  1. 进入nginx配置文件夹 cd ~/nginx-1.11.4/new-deploy/conf (/new-deploy为手动编译nginx后的文件目录)

  2. mkdir security 保存nginx服务器证书

  3. 获取nginx证书

  4. 配置nginx.conf

  5. ./sbin/nginx -t 检查配置是否正确

  6. ./sbin/nginx -s reload 重启nginx
