11. ubuntu上安装QQ

wineQQ更新地址：http://phpcj.org/wineqq
感谢yuking（winelibs20171218）、小俊(9.0.3/23729绿色版）

使用步骤（只需两步，不分先后）：

步骤一、安装wine（详见：https://www.winehq.org/download）
// ubuntu/ubuntukylin/mint（需要加ppa源，再安装）
sudo add-apt-repository ppa:wine/wine-builds;sudo apt-get update;sudo apt-get install winehq-devel
// openSUSE_42.3（需要加源，再安装）
sudo zypper ar http://download.opensuse.org/repositories/Emulators:/Wine/openSUSE_42.3/ Wine;sudo zypper in wine
// arch（直接安装）
sudo pacman -S wine
// gentoo（直接安装）
sudo emerge wine
// Fedora（直接安装）
sudo dnf install wine

步骤二、将压缩包解压到用户主目录（解压出来的是2个隐藏目录和本说明文件）
~ $tar xvf wineQQ9.0.3_23729.tar.xz -C ~/

完成上面两步，点击系统菜单，应该就有了Wine QQ的菜单项。

问题：
1、无法记住密码；
2、无法打开群文件。

13. ubuntu上安装Shadowsocks-qt5

1. shadowsocks-qt5

  • sudo add-apt-repository ppa:hzwhuang/ss-qt5
  • sudo gedit /etc/apt/source.list.d/hzwhuang-ubuntu-ss-qt5-bionic.list (若为ubuntu1804, 则将bionic改成xenial即可, 若为ubuntu1604则忽略该步骤.)
  • sudo apt-get update
  • sudo apt-get install shadowsocks-qt5

2. 配置pac系统全局代理

genpac是一个基于gfwlist的代理自动配置文件生成工具. (gfwlist>pac等各种代理自动配置文件格式)

  • sudo apt install python-pip

  • sudo pip install genpac

  • sudo pip install --upgrade genpac

  • cd /usr/local/etc

  • sudo mkdir ss-pac

  • sudo touch user-rules.txt

  • sudo genpac --pac-proxy="SOCKS5 127.0.0.1:1080" --output="/usr/local/etc/ss-pac/autoproxy.pac" --gfwlist-url="https://raw.githubusercontent.com/gfwlist/gfwlist/master/gfwlist.txt" --user-rule-from="/usr/local/etc/ss-pac/user-rules.txt"

  • 进入设置>网络>网络代理>选择自动>填入pac文件位置: file:///usr/local/etc/ss-pac/autoproxy.pac

  • chrome插件switchyOmega选择系统代理方式.

3. 配置switchOmega插件的情景模式(可选, 在不使用上述系统全局代理方式情况下, 可是用该情景模式)

  • 添加一个情景模式test-ss, 配置代理服务器(代理协议socks5,代理服务器127.0.0.1,端口1080)
  • 在切换规则中, 将规则列表规则选择情景模式test-ss, 规则列表选择autoproxy, 填写gfwlist-url:https://raw.githubusercontent.com/gfwlist/gfwlist/master/gfwlist.txt, 立即更新!
  • chrome插件switchyOmega选择切换规则方式.
