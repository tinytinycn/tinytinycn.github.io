fix-ubuntu连接android手机

May 28, 2019

前提条件

sudo apt install android-tools-adb
lsusb
cd ~/.android
touch usb_adb.ini
sudo vi usb_adb.ini

sudo adb kill-server
sudo adb start-server
sudo adb devices

fix issue

  • sudo vi /etc/udev/rules.d/51-android.rules

  • add this content

  • sudo udevadm control --reload-rules

  • 重新拔插usb

  • adb devices
